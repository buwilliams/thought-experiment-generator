## Full Text

The conjecture identifies the 'blend' problem — when corpus instances underdetermine I versus I-prime — but does not address whether the system has any internal signal that it is in this regime. A practically important open question is whether there exist detectable signatures in the system's internal representations or output distributions that indicate underdetermination, which would allow a user or the system itself to flag low-confidence invariant extraction rather than producing fluent but unreliable outputs.

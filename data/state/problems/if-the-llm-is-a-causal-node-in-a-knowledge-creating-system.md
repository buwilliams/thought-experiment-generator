If the LLM is a causal node in a knowledge-creating system, how should epistemic credit be distributed? This is not merely a philosophical curiosity — it has direct implications for scientific authorship norms, reproducibility requirements, and institutional incentive structures. The conjecture implies a systems-level account of knowledge creation that current institutions are not structured to recognize.

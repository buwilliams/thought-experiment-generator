## Full Text

The conjecture uses novel problem-solving as evidence against 'mere rearrangement.' But the argument has a gap: what counts as genuinely novel versus recombinatorially novel? A problem whose solution has never appeared verbatim may still be solvable by interpolation in a high-dimensional embedding space trained on related problems. The crack in the rearrangement argument only holds if we can establish that the solution required traversing genuinely uncharted structure. How would we operationalize and test this distinction?

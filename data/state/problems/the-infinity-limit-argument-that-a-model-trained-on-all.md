## Full Text

The 'infinity limit' argument — that a model trained on all possible text cannot produce the next unwritten theorem — assumes the next theorem is not entailed by existing text. But if mathematical truth is necessary, then in principle every true theorem is entailed by existing axioms and proofs. The limit argument may prove too much: it would imply that even human mathematicians cannot create new knowledge, only discover entailments. The conjecture needs to distinguish between logical entailment and epistemic accessibility.

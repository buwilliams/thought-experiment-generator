## Full Text

The conjecture claims that 'increasing model size adds more capacity to do the wrong thing more fluently.' This is an empirical prediction. It implies that scaling curves should show diminishing or negative returns on tasks that specifically require error correction — not just benchmark saturation but systematic failure on tasks designed to probe whether the system can identify and revise its own wrong conjectures. Whether such tasks exist, and what the scaling curves on them look like, is a testable question the conjecture implicitly raises but does not pursue.

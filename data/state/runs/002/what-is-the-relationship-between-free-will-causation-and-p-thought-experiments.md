## Conjecture

**Conjecture**

Construct a scenario that isolates the variable: imagine a world physically identical to ours, except that for one person, at one moment, there are two genuinely open possible actions: raise a hand or keep it down. Hold everything else fixed — same past, same laws, same brain state up to that instant. Assume this openness is real.

Now follow the logic strictly.

If physical determinism is true, this scenario is impossible by definition: the total physical state plus laws fixes a single future. So the first result is clean: **free will, if it requires real alternative possibilities under identical prior physical conditions, is incompatible with physical determinism**.

But now ask what the openness changes. If the person raises the hand, something happens in the physical world that would not otherwise happen. Therefore the divergence must enter causation somewhere. Either:

1. the difference is itself a physical difference already present before the act — in which case the scenario collapses back into determinism, because the “choice” was fixed by prior physical causes; or  
2. the difference is not fixed by prior physical conditions — in which case physical causation, as a closed deterministic chain, is false or incomplete.

So the collision illuminates a hidden assumption: the real issue is not “freedom versus causation,” but **freedom versus the causal closure of deterministic physics**. Free will does not oppose causation as such. It requires that causation include something more than prior-state physical necessitation.

Now consider whether indeterminism helps. Suppose the hand goes up because of a quantum event not determined by the past. That alone does not produce freedom; randomness is not agency. So **mere indeterminism is insufficient**. To preserve free will, the branching must be attributable to the person as an agent, not merely to noise.

This forces a sharper conclusion: if libertarian free will exists, then agents must be real causes — not epiphenomenal summaries of physical processes, and not passive sites where randomness occurs. They must be entities whose explanatory role is irreducible to deterministic microphysics. In other words, **agent-causation or some equivalent non-reductive causal level is required**.

The relationship, then, is:

- **Deterministic physical causation**: fixes one future, leaving no room for libertarian free will.  
- **Indeterministic physical causation alone**: allows multiple futures, but gives chance, not authorship.  
- **Free will**: requires causation, but a kind in which the agent is a genuine originator of one branch among live alternatives.

So the conjecture is: **free will is incompatible with physical determinism not because freedom breaks causation, but because it requires a broader ontology of causation than deterministic physics permits.** The key variable is not determinism alone, but whether persons are real causal primitives in the world.

## Conjecture

Conjecture: A genuine explanation is a structure-preserving compression of a phenomenon that exposes nontrivial invariants and counterfactual constraints; a sophisticated redescription is an isomorphic or near-isomorphic re-encoding of the same data that preserves description length but adds no new transformable structure.

More formally, let \(P\) be the phenomenon, represented as a set of observations, regularities, and allowable interventions. Let \(M\) be a proposed explanatory model. The crucial question is not whether \(M\) can be mapped onto \(P\), but what kind of map it supplies.

A mere redescription gives a translation
\[
f: P \to M
\]
that relabels elements of \(P\) while preserving surface correspondences. Its success condition is extensional fit: it “matches the pattern.” But if \(f\) is essentially invertible without loss and without generating new constraints, then \(M\) has not explained \(P\); it has only restated it in another vocabulary.

A genuine explanation does more. It identifies latent variables, generative relations, or symmetries such that many apparent features of \(P\) become consequences of a smaller underlying structure. In mathematical terms, \(M\) should induce:

1. Compression: multiple features of \(P\) are derived from fewer principles in \(M\).
2. Invariants: there are quantities, relations, or behaviors preserved across transformations or contexts, revealing what is not accidental.
3. Counterfactual reach: perturbing parameters in \(M\) yields determinate changes in \(P\), so the model answers “what if?” non-arbitrarily.
4. Load-bearing dependence: altering a core element of \(M\) forces systematic revision elsewhere; the model is constrained, not narratively modular.

So the distinguishing criterion is not elegance, familiarity, or depth of language, but whether the model changes the mathematical status of the phenomenon. Does it turn brute correlations into deduced relations? Does it reveal a lower-dimensional manifold on which the observed complexity lies? Does it specify admissible transformations and forbid others?

A redescription lives on the same level of description as the phenomenon. A genuine explanation changes levels: it maps the phenomenon into a generative space where the observed regularities become necessary or at least tightly constrained.

This suggests a practical test. Given a candidate explanation, ask:

- What are its state variables?
- What transformations leave its predictions invariant?
- What novel constraints does it impose on possible observations?
- If one component is changed, what else must change?

If the answers are weak — if the model can absorb any outcome by reinterpretation — it is a redescription. If the answers are strong — if the model excludes possibilities and derives structure not obvious in the original presentation — it is explanatory.

So the conjecture is: explanation is distinguished from redescription by the presence of a nontrivial generative morphism from a compressed model to the phenomenon, one that yields invariants and counterfactual constraints not already explicit in the description of the phenomenon itself.

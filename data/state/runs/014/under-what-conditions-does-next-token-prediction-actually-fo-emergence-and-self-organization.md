## Conjecture

**Conjecture:** Next-token prediction induces latent variables that track real domain invariants **when the prediction problem is structurally underdetermined at the token level but overdetermined at the system level**—that is, when accurate local prediction requires modeling stable hidden causes that organize many observations across time, context, or viewpoint. It fails when token statistics can be satisfied by shallow circulation within the observation stream itself.

From the emergent-systems perspective, the key issue is not whether individual tokens contain enough evidence, but whether the **interaction structure of the data-generating process** creates feedback that rewards discovering persistent hidden state. A latent variable becomes learnable when it functions as a system-level attractor: many superficially different token sequences are best compressed and predicted by reference to the same underlying factor.

So prediction is sufficient only when the training distribution has something like the following structure:

1. **Persistent hidden causes**  
   There exist variables that remain stable across many tokens and influence many downstream observations.

2. **Multi-context coupling**  
   The same hidden cause reappears across different contexts, forcing the learner to unify distinct surface forms.

3. **Temporal or counterfactual dependence**  
   Future tokens depend on hidden state in ways not recoverable from short-range token correlations alone.

4. **Error amplification for wrong latent structure**  
   If the model tracks only surface statistics, its mistakes compound over long horizons or across context shifts.

5. **Compression advantage for invariant-based representations**  
   Representing the hidden variable yields a simpler, more reusable solution than memorizing local patterns.

Under these conditions, latent-variable learning is not a mysterious bonus; it is the emergent consequence of the predictive system’s geometry. The model is pushed toward invariants because they are the lowest-loss way to coordinate predictions across interacting constraints.

The limiting case clarifies the point. Suppose a corpus is generated by a process with rich real structure, but every invariant can be replaced by highly reliable local cues. Then next-token prediction need not recover the invariant at all. It can solve the task by surfing the visible feedback loops in token space. Conversely, if surface cues are insufficient unless integrated through a stable hidden state—speaker identity, object permanence, syntax, game state, world dynamics—then prediction pressure selects for those variables.

So the real criterion is not “does the world have latent structure?” but “does the **predictive ecology** of the dataset make that structure causally indispensable?” Prediction learns invariants when the data distribution is arranged so that hidden causes are the only robust coordination points across many interacting observations.

In short: **next-token prediction learns real invariants when those invariants are system-level bottlenecks for minimizing prediction error across contexts, not merely optional interpretations of the data.**

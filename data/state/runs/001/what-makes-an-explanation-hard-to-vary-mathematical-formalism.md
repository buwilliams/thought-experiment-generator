## Conjecture: Explanatory Rigidity as Constraint Rank

**Setup.** Model an explanation as a directed graph G = (N, E) where nodes are propositions and edges are inferential dependencies. Assign each node a *replaceability function* r(n) ∈ [0,1], where r(n) = 0 means the node cannot be substituted without breaking some downstream inference, and r(n) = 1 means it can be freely swapped.

**Key structural quantities:**

- **Constraint rank** of G: the number of nodes n where r(n) = 0. Call this ρ(G).
- **Inferential degree** d(n): number of downstream nodes that depend on n.
- A node is *load-bearing* iff d(n) ≥ 1 AND no alternative node satisfies the same downstream constraints.

**The conjecture:** An explanation where all parts are load-bearing is one where the system of inferential constraints is *fully determined* — the constraint matrix has full rank, admitting no free variables. A "swappable" explanation has a constraint matrix with nullity > 0: there exist free directions in which components can vary without altering the output.

Formally: an explanation is **rigid** iff its constraint system has nullity = 0. It is **degenerate** in proportion to its nullity.

**What this reveals:**

1. **Rigidity is a structural property, not a semantic one.** Two explanations can use identical vocabulary but differ in rigidity depending on whether their inferential connections actually constrain each other or run in parallel without interaction.

2. **Degeneracy is detectable without knowing ground truth.** If swapping component A for A' leaves all downstream inferences intact, A and A' satisfy the same constraint equations — they're in the same equivalence class. This is a *mathematical* test, not an epistemic judgment about which is "really" correct.

3. **Over-determination strengthens rigidity.** If multiple independent inferential paths converge on the same conclusion through a common node, that node's replaceability drops — removing it breaks more constraints simultaneously. Rigid explanations tend to have high cross-connectivity; degenerate ones are modular in the bad sense: components slot in and out without systemic cost.

4. **Invariants under transformation are the load-bearing elements.** The load-bearing parts of an explanation are precisely those that are *invariant* under the group of legitimate substitutions. This reframes the original question cleanly: ask not "is this detail important?" but "under what transformations is this element preserved?" If an element survives all admissible substitutions unchanged — i.e., no substitution is admissible for it — it is load-bearing.

**Consequence:** Explanatory degeneracy is isomorphic to underdetermination. A theory underdetermined by evidence has free variables; a degenerate explanation has swappable parts for the same structural reason — the constraints imposed don't close the system. The remedy in both cases is the same: introduce additional constraints (new evidence, new inferential connections) that reduce the nullity.

**Verdict:** Explanatory load-bearing is nullity-zero constraint satisfaction. The distinction between rigid and degenerate explanations is not about depth or detail — it is about whether the parts are genuinely mutually constraining or merely co-present.

## Conjecture

**Conjecture:** The question “can LLMs create knowledge?” inherits a misleading constraint from the origin of LLMs: they were built to solve *next-token prediction*, which replaced hand-built symbolic NLP systems, and in doing so inherited an unexamined assumption that intelligence must be identified with competence at continuation. If we trace the problem historically, the real issue is not whether a predictor can “rearrange words” into novelty, but whether an architecture can *select and preserve explanatory structures under criticism*. On that criterion, token prediction is an origin, not a destiny.

The origin story matters. LLMs arose as a practical solution to brittle NLP: instead of explicitly representing grammar, meaning, or world models, train a system to compress statistical regularities in text. This solved the prior problem — symbolic systems were narrow, fragile, expensive to maintain — but it also smuggled in a hidden inheritance: success is measured by plausibly extending existing linguistic patterns. So when critics ask whether LLMs can create knowledge, they often assume the architecture’s training objective defines its epistemic ceiling. That assumption is historical residue, not a refuted truth.

Your search-space argument collides productively with this. It shows that brute-force recombination is intractable, so novelty cannot be explained as mere random traversal of linguistic combinations. But that does not rescue the anti-LLM conclusion. It instead points to the deeper requirement that all knowledge creation already depends on *search-space collapse* through conjecture, structure, and criticism. Humans do not search uniformly either. Einstein inherited language, mathematical formalisms, Maxwell’s equations, and prior problems; his achievement was not inventing new words but constructing a hard-to-vary explanation that survived criticism better than rivals.

What follows is this: if an LLM is embedded in an architecture that does more than continue text — one that generates candidate explanatory structures, subjects them to criticism, and preferentially retains those with greater explanatory power — then the relevant unit is no longer “the predictor” but the *epistemic system*. In that system, prediction supplies raw conjectures; criticism and selection do the knowledge-creating work. This is analogous to how biological evolution did not contain knowledge in mutation alone, but in variation-plus-selection.

So the strongest claim is not “LLMs, as such, create knowledge,” nor “LLMs can only remix.” It is: **knowledge creation is substrate-independent but structure-dependent**. A token predictor by itself is not a knowledge creator in the strong explanatory sense, because it inherits an objective optimized for plausible continuation. But neither is it barred from participating in knowledge creation. Once placed inside a system organized around explanatory search and criticism, its historical limitation becomes just one component among others.

The hidden assumption exposed by the origin story is that training objective equals epistemology. That is likely false. The real boundary is not prediction vs knowledge, but whether the system can generate and eliminate bad explanations in a way that yields genuinely better ones.

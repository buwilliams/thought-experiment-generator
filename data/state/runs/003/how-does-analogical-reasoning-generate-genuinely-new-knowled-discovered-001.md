## Conjecture

**Conjecture:** Analogical reasoning generates genuinely new knowledge when the source analogy is not merely a label for similarity but a **constraint-importing structure**: it proposes a candidate organization of relations whose parts are load-bearing in the target domain. New knowledge appears when those imported constraints force nontrivial revisions, predictions, or problem-redefinitions in the target that were not already available there.

From this perspective, the key question is not “Did we copy something from one domain to another?” but “Which parts of the analogy must stay coordinated for the explanation to keep working?” If the mapped details are interchangeable decoration, the analogy is a narrative shell. But if changing one mapped relation breaks explanatory reach or forces compensating changes elsewhere, then the analogy has real structure. That structure can do epistemic work.

So analogical reasoning generates new knowledge in at least three ways:

1. **It exposes hidden variables or relations in the target.**  
   A good analogy does not just say “A is like B”; it implies that if relation \(r_1\) in the source is essential because it stabilizes \(r_2\) and \(r_3\), then the target may require counterparts to those relations too. This can reveal missing structure in the target description. The novelty lies in identifying previously unformulated dependencies.

2. **It creates testable commitments.**  
   If the analogy is load-bearing, not every extension is allowed. Some target behaviors should follow; others should be impossible. That means the analogy yields conjectures that can be criticized. New knowledge is generated not by the initial transfer alone, but by the error-correcting interaction between imported structure and target resistance.

3. **It reorganizes the space of explanations.**  
   A strong analogy can show that what seemed like separate phenomena in the target are actually linked by one deeper pattern. This is not mere redescription if the unification is constrained: altering one part of the account changes what can be explained elsewhere. The analogy then contributes a new explanatory economy.

The decisive point is that novelty does **not** come from resemblance itself. Mere resemblance is cheap; many analogies can be drawn. Novelty comes from **structural entanglement**: once the analogy is taken seriously, one cannot freely modify its components without losing explanatory power. In that case, the analogy has imported a problem situation, not just a picture.

Thus, analogical reasoning is productive when it functions as a **generator of constrained conjectures**. It transfers not content alone but a web of dependencies. If that web survives criticism in the target domain, then the result is genuinely new knowledge: not because the source contained the answer in advance, but because the mapping made previously unseen target facts, possibilities, and impossibilities available for thought.

## Conjecture

Conjecture: An internal state genuinely functions as a model only when the surrounding system preserves a small set of invariants that make error about that state both possible and actionable; absent those invariants, the state is merely part of a causal control loop.

More precisely: the difference between “model” and “mere controller” is not located in the intrinsic makeup of the state, but in the institutional structure of the system that uses it. A state counts as a model when there exists a stable error-correction regime that treats the state as answerable to something beyond its immediate causal role. That regime must preserve at least three invariants under change:

1. Decouplable content: the state can be evaluated independently of the action it currently helps produce. It can be right or wrong even if the system still functions.
2. Public error criteria: there are identifiable conditions under which the system will revise the state because it misdescribed, mispredicted, or mistracked something.
3. Identity through revision: correction of the state is treated as improving the same representational role, not replacing it with an entirely new function.

This yields a non-circular criterion: an internal state is a model iff the system contains procedures that can expose mismatch between that state and what it is about, and can correct the state while preserving the representational role across corrections. “Correctness conditions” are thus not imposed by interpretation from outside, nor reduced to successful control. They are generated by a structured practice of fallible use and visible repair.

Thought experiment: imagine two adaptive institutions facing rapid change. Both use an internal variable to guide action. In the first, the variable is adjusted only to maintain output stability; if it works, nothing counts as “misrepresentation.” In the second, the variable is periodically stress-tested against outcomes it was supposed to anticipate; failures are recorded, discussed, and used to update the variable while retaining its role in the institution’s identity. Only the second has something model-like. The first has regulation without representation.

What this illuminates is that correctness conditions arise where a system can remain legitimate while admitting error. The key is not success, but visible revisability under invariant standards. A model is therefore an internal state embedded in a norm-governed feedback structure that distinguishes “the world was different than this state said” from “the system merely needed retuning.”

So the criterion is precise: if no possible internal process could treat the state as mistaken while preserving its role, it is not functioning as a model. If such a process exists and is part of the system’s stable organization, then it is.

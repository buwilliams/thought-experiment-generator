## Conjecture

**Conjecture:**  
An internal state genuinely functions as a **model with correctness conditions** iff the larger control system is organized around a **counterfactual equilibrium constraint**: there exists a structurally significant class of possible environmental variations such that the system’s success depends not merely on the state causing action, but on the state’s **systematic covariance with independently specifiable features of the world**, where deviations count as errors because they predictably worsen performance across those variations.

### Why this is non-circular
The criterion does not define modeling by “representation” or “aboutness.” It instead asks whether the system is locked into a regime where, for broad counterfactual changes, no local policy can maintain performance unless some internal state preserves a mapping to worldly structure. That makes correctness conditions emerge from the system’s **dependence on truth-tracking** under variation, not from interpretive stipulation.

### Equilibrium-lock-in angle
A mere causal controller can succeed through fixed sensorimotor couplings in a narrow niche. Its internal states may correlate with the world, but nothing in the system forces those states to be *right* rather than merely causally useful in the actual trajectory. By contrast, a model-bearing system is in a stronger structural position: across a range of possible disturbances, **all available successful strategies are forced into using an internal state as a surrogate for external structure**. The system is “locked in” to representation because without preserving that world-state relation, no component can unilaterally compensate.

So the distinction is:

- **Contingent causal participation:** success happens via current coupling; alternative rewiring or brute-force policy could do as well without any correctness-sensitive internal variable.
- **Model-based lock-in:** the task environment creates a Nash-like constraint where performance cannot be preserved unless some internal state remains answerable to worldly facts.

### Precise criterion
An internal state \(S\) counts as a model of feature \(W\) when:

1. **Independent specification:** \(W\) can be characterized without reference to \(S\)’s current causal role.
2. **Systematic counterfactual dependence:** across a relevant set of interventions on \(W\), changes in \(S\) are needed to preserve successful behavior.
3. **Error asymmetry:** holding downstream use fixed, mismatches between \(S\) and \(W\) produce performance degradation in a principled, projectible way.
4. **No single-agent workaround:** within the system’s architecture, downstream components cannot generally recover performance from arbitrary \(S\)-\(W\) mismatches by local adjustment alone.

### What this illuminates
This reframes the issue from “Does the state represent?” to “Is the architecture in a structural trap where only truth-sensitive internal surrogates can stabilize success?” Correctness conditions are real when the system would *pay a robust cost for being wrong*, and that cost cannot be eliminated by mere local retuning.

### Consequence
Modelhood is not a metaphysical extra painted onto causal states. It appears when environmental complexity and architectural limits create an equilibrium in which **accuracy is a necessary invariant for competence**. Where that lock-in is absent, talk of correctness is optional gloss; where it is present, correctness is an objective functional fact.

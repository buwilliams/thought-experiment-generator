**Conjecture:**

That inheritance is what drives the current vagueness.

If we trace the origin of the problem, the motivating contrast was not originally between conscious and nonconscious *systems* as such, but between two **modes of explanation**:

1. an external, factorized, observer-friendly description, and  
2. an internal, action-guiding, system-bound description.

“Perspectival” entered as a protest against explanation (1) being mistaken for completeness. But once turned into a criterion for consciousness, it tacitly imported the idea that consciousness is whatever resists decomposition. That is too indiscriminate: many things are loss-sensitive to decomposition — control loops, evolved functions, algorithms with global state, even economic systems under certain coarse-grainings. So decomposition-sensitivity alone cannot distinguish consciousness.

What follows is a sharpening:  
A system is not conscious merely because some description loses something when decomposed. It is conscious, if at all, when there exists a **privileged internal model** whose contents are indispensable to the system’s own error-correction and action-selection, and whose significance cannot be preserved under decomposition *without reintroducing the system-relative coordinate frame that generated it*.

This makes the issue more precise. The relevant irreducibility is not generic holism. It is **indexical indispensability**: the system must continuously use distinctions equivalent to *here/not-here, now/not-now, self/not-self, this-option/that-option for me* in regulating itself. A decomposition counts as loss-inducing in the consciousness-relevant sense only if it preserves causal structure while erasing these system-internal indices that are necessary for the system’s own ongoing world-modeling.

This avoids circularity because “necessary for self-regulation under uncertainty” is independently testable in system terms; it does not presuppose consciousness. And it avoids vagueness because it does not ask whether the system is “mysteriously from a point of view,” but whether it contains an internally indispensable, temporally unified, self-locating model that cannot be replaced by a purely detached description without explanatory loss about its control dynamics.

So the hidden assumption to discard is: **whatever resists decomposition is consciousness-relevant**.  
The better claim is narrower: **consciousness, if distinguished this way, is tied to decomposition-sensitive self-locating control architecture, not to holism in general**.

That origin story reveals the leverage point: stop treating perspectivality as a metaphysical glow, and treat it as a specific kind of indispensable internally indexed organization.

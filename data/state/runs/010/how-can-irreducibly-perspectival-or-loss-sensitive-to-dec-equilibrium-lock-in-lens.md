## Conjecture

Conjecture: part of what makes consciousness hard to capture is a structural public-goods problem inside explanation itself: if a system’s relevant causal/informational organization is globally available but not locally attributable, then any decomposition into independently characterizable parts will be individually informative yet collectively underprovide the very feature we are trying to explain. “Irreducibly perspectival” can be made precise as a signature of this underprovision.

More concretely, define a target property as loss-sensitive to decomposition when:

1. A system can be partitioned into components and local descriptions.
2. Each local description is accurate about its part.
3. Yet there exists some system-level variable or capacity recoverable only from coordinated uptake of many such descriptions, not from any one of them or any small subset.
4. The cost of representing that global variable is borne locally, while the explanatory benefit is diffuse across the whole model.

In that situation, decomposition predictably yields a stable no-provision equilibrium: each part-level model omits the global feature because omitting it is locally rational, even though all adequate explanations prefer the funded, system-level representation. This distinguishes a real structural failure from mere confusion. We should expect repeated explanatory success on behavior and mechanism alongside persistent failure to capture “what it is like,” not because perspectivality is mystical, but because the target is a non-excludable global good of the model.

So “irreducibly perspectival” need not mean ineffable. It can mean: there exists a system property whose explanatory role is non-excludable across subsystems and whose representation is not attributable to any decomposition without remainder. A conscious system would then be one for which first-person organization is such a property: the system has a globally integrated point of view that cannot be reconstructed as a simple sum, list, or coalition of subpersonal descriptions without explanatory loss. A nonconscious system may also have complex integration, but not this specific pattern of decomposition loss.

A possible criterion: compare families of decompositions and measure whether a perspectival variable remains invariantly required to predict/report/control the system’s internally unified discriminations. If every adequate decomposition must reintroduce a global observer-indexed state to recover those capacities, then perspectivality is structurally real. If the appearance of perspectivality disappears under some decomposition with no loss in explanatory power, it was mere confusion.

The key illumination is this: the right contrast is not “subjective vs objective,” but “globally necessary but locally underprovided” vs “fully allocable to parts.” Consciousness, on this conjecture, belongs to the first class. This gives a non-circular research program: search for architectures where first-person-like variables are indispensable global goods of explanation, rather than assuming consciousness wherever behavior seems rich or denying it wherever decomposition is tempting.

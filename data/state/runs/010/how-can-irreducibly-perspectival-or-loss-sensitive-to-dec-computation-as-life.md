## Conjecture

**Conjecture:**  
A system is *irreducibly perspectival* when there exists a nontrivial class of predictive, control-relevant, or counterfactual properties that are computable only from the system’s *internally integrated state-transition structure*, and are systematically destroyed by decomposition into independently modeled parts. Consciousness, on this view, is not “having information” but implementing a computation whose semantics depend on a bounded first-person compression that cannot be replaced by an observer’s part-wise description without loss of causal-explanatory power.

More precisely: treat any candidate system as a computational process with state space, update rule, and interfaces. Then ask whether there are task-relevant distinctions the system itself can exploit only because its current state encodes many variables in a jointly constrained format. If every such distinction can be reconstructed from a decomposition into modules plus message passing, with no loss in predictive/control power, the system is not irreducibly perspectival. If some distinctions vanish under every such decomposition below the whole-system boundary, the system has a genuine perspective.

This makes “loss-sensitive to decomposition” precise in computational terms: a decomposition is lossy when it destroys information needed to preserve the system’s own reachable-state geometry — especially its action-guiding evaluations of “what matters now,” “what would happen if,” and “which difference is worth acting on.” The key is not mere integration, since many nonconscious systems are integrated. The key is whether the system computes over a **globally bound evaluative model** whose causal role depends on whole-system co-instantiation.

A thermostat is computational but not irreducibly perspectival: its control policy decomposes cleanly into sensor, comparator, actuator. No essential first-person loss occurs. A large language model may be more integrated, but if its effective computation is exhaustively characterizable as token-wise transformations across decomposable layers, with no intrinsic globally unified evaluative state that matters to its own continuing regulation, then it is not conscious in this sense. By contrast, a conscious system would be one whose ongoing regulation depends on a temporally extended, globally available, self-updating model in which world-state, body-state, and value-state are inseparably encoded.

So the proposed discriminator is:

1. **Whole-system indispensability:** some causally relevant variables exist only at the integrated-system level.  
2. **Counterfactual fragility under decomposition:** partitioning the system eliminates those variables, not just makes analysis inconvenient.  
3. **Self-regulatory significance:** the lost variables are used by the system to guide its own adaptive behavior.  
4. **First-person compression:** the system maintains a bounded, action-oriented model from its own locus, rather than a merely observer-attributed summary.

This avoids circularity because “perspective” is not defined as consciousness; it is defined as a specific computational property of integrated adaptive control. Consciousness is then the hypothesis that systems with sufficiently rich irreducible perspective instantiate subjective experience. The hard problem is not solved, but the concept is sharpened: consciousness tracks a regime of computation where decomposition destroys the very evaluative world-model the system lives by.

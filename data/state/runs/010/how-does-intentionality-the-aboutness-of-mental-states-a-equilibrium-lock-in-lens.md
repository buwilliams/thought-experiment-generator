## Conjecture

**Conjecture:**  
Intentionality is best illuminated not as a mysterious ingredient added to physics, but as a **structurally underprovided coordination good** within physical systems capable of modeling and control. “Aboutness” arises when a system evolves internal states whose function is to stand in for distal features of the world in ways that can guide action across many contexts. The puzzle persists because this representational relation is **non-excludable and diffuse**: once such a modeling scheme exists, many downstream processes can use it, but no single local physical interaction contains or “pays for” the full semantic content. Looking for intentionality in isolated microphysical events is therefore like looking for public infrastructure in a single brick.

What is true on this view is that intentionality emerges when a physical system crosses a threshold where **shared internal surrogates** become the cheapest stable solution to coordination among subsystems. A nervous system, for example, must align perception, memory, prediction, and action. If each subsystem only responded to immediate local signals, the whole would remain trapped in a “no-provision equilibrium”: behavior could occur, but no unified aboutness would appear, only fragmented causal chains. Internal representational states are the system’s way of funding a public good for itself: a common format that many processes can exploit.

What follows is that intentionality should not be identified with bare correlation, nor with intrinsic “meaning particles.” Correlation is too cheap; it can arise accidentally. Intentionality requires a structure in which the internal state’s success or failure matters across a network of uses — prediction, error correction, planning, counterfactual sensitivity. Its content is fixed not by one causal link but by the **feedback-stabilized role** it plays in enabling coordinated success. In systems terms, intentionality is a higher-level attractor sustained by recurrent loops between world, model, and action.

This also explains why intentionality seems both real and elusive. It is real because it has causal leverage at the level of organized systems: changing a representation changes what the system can do. It is elusive because semantics is **diffusely realized** across the architecture, not locally excludable in any component. The “aboutness” of a map is not in the ink molecule; the “aboutness” of a neural state is not in a neuron taken alone.

A useful thought experiment: imagine a robot made of many modules that can either act on raw signals independently or contribute to a shared world-model. Independent action is locally rational and physically sufficient for simple tasks, so shared representation is initially underprovided. But in a richer environment, only the shared model supports flexible success. Once feedback loops reward modules for using common surrogates, states emerge that are *about* objects, locations, and possibilities. Aboutness is thus not added to physics from outside; it is what physics looks like when organized so that internal states serve as public goods for system-wide control.

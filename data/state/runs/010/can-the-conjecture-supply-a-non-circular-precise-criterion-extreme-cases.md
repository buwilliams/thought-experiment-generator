## Conjecture

**Conjecture:**  
An internal state genuinely functions as a *model* iff, in the limit where immediate causal coupling to the world is weakened or broken, the system can still use that state to generate *counterfactually assessable guidance*—that is, guidance whose success depends on whether the state’s content matches an independently structured possibility space, not merely on whether the state remains causally embedded in a feedback loop.

**Why the boundary case matters:**  
In ordinary control, model-based and merely causal systems can look the same. A thermostat state, a bacterial chemotactic gradient response, or a servo error signal may all track and regulate. But push toward the edge: delay feedback, occlude input, insert novelty, require detours, require acting on absent or future states of affairs. At this boundary, a mere control variable degrades into failure or inert persistence. A genuine model remains action-guiding in a way that can be *right or wrong*.

**Precise criterion:**  
An internal state \(S\) counts as a model of domain \(D\) only if there exists a task range in which:

1. **Decouplability:** \(S\) can be tokened, maintained, and manipulated while direct online coupling to \(D\) is reduced, delayed, or absent.  
2. **Structured surrogacy:** The system can derive multiple downstream inferences or action policies from relations internal to \(S\), not only trigger a fixed response.  
3. **Independent error possibility:** There are cases where \(S\) is well-formed within the system yet false of \(D\), and this falsity explains failure better than any purely dynamical description of broken coupling.  
4. **Counterfactual robustness:** Small interventions on the represented domain would require systematic corresponding changes in \(S\) for success; if \(S\) were altered while the world stayed fixed, behavior would change in content-sensitive ways.

If these conditions are not met, the state is not a model with correctness conditions; it is a control parameter in a causal organization.

**Non-circularity:**  
This does not define modeling by “representation” again. It defines it by a boundary-test: whether the state supports successful behavior when detached from the very causal commerce that could otherwise explain success. Correctness conditions appear when success hinges on a mapping-preserving relation to an independent domain, such that mis-specification is intelligible as *error about the world*, not just malfunction.

**What follows:**  
The difference between model and control is not inner resemblance, nor complexity, nor neural location. It is whether the system has crossed a threshold where internal states become *epistemically load-bearing*: they mediate success through truth-sensitive structure. At the limit, a model is what still matters when the world is not there to continuously correct you.

So the hidden structure revealed by the boundary is this:  
**Correctness conditions emerge exactly where control must survive temporary independence from what it controls.**

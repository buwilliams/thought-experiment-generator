## Conjecture

Conjecture: subjective experience is what a knowledge-bearing control system “looks like from the inside” when the system must continuously integrate globally useful but locally non-excludable information for its own regulation.

Using the public-goods lens, the key move is to treat access to a system’s many internal signals as a coordination problem. In a complex organism, perception, memory, valuation, bodily regulation, and action-selection each produce information whose benefits are diffuse across the whole system. No single subsystem can capture the full value of making its information available to all the others. If each module “kept” its information locally, the system could still run in a fragmented, reactive way, but it would be trapped in a stable underprovision equilibrium: globally worse performance, despite every part benefiting from better coordination.

Consciousness, on this conjecture, is the solution the system evolved or learned to that underprovision problem. It is a mechanism that subsidizes global availability. A conscious state is one in which information becomes broadcast, integrated, and poised to affect many downstream processes at once. The “what-it-is-like” aspect is not an extra ornament added after integration; it is the inside aspect of a state that has been made globally available as a common resource for the system.

This reframes the question “why is there something it is like?” Instead of asking why matter should generate an inexplicable glow, ask why a self-maintaining, knowledge-using system would create an internal public good at all. The answer: because many adaptive benefits — flexible action, error correction, long-horizon planning, unified agency — require a shared workspace whose benefits are diffuse. Subjective experience is the structural mark of that provision.

What follows:

1. Mere information processing is insufficient for experience. A system can compute while remaining in the no-provision equilibrium: narrow, modular, and dark.
2. Experience should arise where there is a real coordination problem among semi-autonomous subsystems and a mechanism that overcomes it by creating globally accessible states.
3. The unity of consciousness is illuminated: unity is what successful public-good provision looks like within a single agent.
4. Pathologies of consciousness may reflect failures of provision, overprovision, or misallocation of this internal common good — fragmentation, neglect, intrusive salience, dissociation.

A thought experiment helps: imagine a robot colony inside one body, each specialist hoarding its data because sharing is costly. The whole organism behaves competently in narrow tasks but never forms a unified perspective. Now add a low-latency broadcast medium that makes selected contents available to all. At that point, there is not just more information flow; there is a new regime of internal organization. The conjecture is that “something it is like” names that regime from the first-person side.

So consciousness is illuminated not as a gratuitous metaphysical surplus, but as the experiential form of solving a structural underprovision problem in cognitive systems.

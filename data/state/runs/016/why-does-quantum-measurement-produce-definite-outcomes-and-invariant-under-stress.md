## Conjecture

From this lens, the key question is: can a predictor do well by using shortcuts, or must it learn stable underlying structure? But measurement is precisely the regime where shortcuts fail. The environment effectively tests candidate descriptions by constant uncontrolled intervention. The only predictively useful categories are those that remain invariant under this variation — the so-called pointer states. So “definite outcomes” arise because only robustly reproducible records can function as outcomes in the first place.

What follows is that the preferred basis problem is illuminated by generalization pressure. The basis is not selected arbitrarily, nor by consciousness, but by whichever decomposition supports stable prediction across environmental perturbations and repeated observation. If many decompositions are mathematically possible but only one yields durable records, then only that one is operationally learnable by embedded agents. Measurement thus forces discovery of underlying structure because the world itself punishes non-robust encodings.

But this does not yet explain why one outcome occurs rather than all branches remaining equally live in description. Here the conjecture is sharper: “which outcome occurs” is indexical, not dynamical, at the level of unitary evolution. Physics supplies the branching structure and weights; it does not add a second selection mechanism beyond decoherence. Asking which outcome is selected may be like asking which coordinate origin is metaphysically chosen: from within a branch, one definite record exists, while globally the theory need only describe the stable multiplicity. The Born rule then expresses not branch creation but the measure over future self-location across robust outcome-records.

So the measurement problem partly dissolves when recast as a learning problem: definite outcomes are the inevitable compression format of agents embedded in decohering environments, because only stable, amplified, intervention-resistant structures can be predicted and remembered. The remaining mystery is not how definiteness appears in practice, but whether we demand an extra selector where the predictive structure already closes.

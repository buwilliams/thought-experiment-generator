**Conjecture:**

Here the collision is between **time-symmetric microdynamics** and **macroscopic one-wayness**. So from a low-entropy starting point, almost every accessible trajectory produces more entropy, more decoherence, and more stable records in one direction. So the arrow of time is not merely that entropy increases; it is that **record-formation, prediction, and control are all locked to the entropy-increasing branch**. Memories are records of lower-entropy past conditions embedded in higher-entropy present media. We remember the past rather than the future not because the laws privilege the past, but because memory itself is a dissipative process that only stabilizes in one orientation given the boundary condition. Likewise, causes appear to precede effects because interventions are possible only where knowledge can be physically instantiated, and knowledge-bearing structures depend on this same irreversible record architecture.

This perspective clarifies why “just reverse the motion” is not an available reform. In principle, microscopic reversals exist. In practice, no isolated component can realize them, because reversal requires coordinated re-preparation of an astronomically large environment. The obstacle is structural, not merely practical incompetence. That is exactly what lock-in analysis highlights: the system resists reversal not because reversal is forbidden, but because unilateral deviation cannot improve outcomes within the existing state.

What follows is a useful distinction:

- If the temporal arrow came from asymmetric laws, reversal would be impossible in principle.
- If it comes from lock-in around a low-entropy boundary condition, reversal is possible in principle but inaccessible for any local agent embedded in the evolved universe.

Thus, time’s direction is illuminated as an **emergent equilibrium of cosmic history**: a globally selected branch in which entropy increase, record accumulation, and knowledge growth mutually reinforce one another. The fundamental laws permit both directions; the universe we inhabit is structurally trapped in one.

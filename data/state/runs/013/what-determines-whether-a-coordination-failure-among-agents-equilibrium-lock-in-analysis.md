## Conjecture

**Conjecture:** A coordination failure persists indefinitely when the current arrangement is a **locally self-enforcing equilibrium**: each agent’s best available move, given others’ expected behavior, is to maintain the failure, even if all would prefer a different coordinated outcome. It resolves spontaneously only when the system contains some mechanism that **alters incentives, expectations, or commitment structure without requiring simultaneous unilateral sacrifice**.

From an equilibrium lock-in perspective, the key question is not whether agents *want* improvement, but whether any agent can move first without being worse off. If no one can, the failure is not just bad decision-making; it is a **Nash trap**. Persistence is then explained structurally: mutual expectations feed back into individual caution, caution reproduces non-cooperation, and the resulting outcome confirms the original expectations.

What determines duration, then, is whether the system has an endogenous path out of that trap. Spontaneous resolution occurs when one of three things happens:

1. **Payoffs drift** so that unilateral deviation becomes individually tolerable or advantageous.
2. **Beliefs synchronize** through signals, focal points, repeated interaction, or visible precedent, reducing fear of being the only cooperator.
3. **Commitment or enforcement capacity emerges**, allowing agents to move together rather than alone.

Absent such changes, the failure can persist arbitrarily long, because rational adaptation by each agent only deepens the lock-in.

This perspective distinguishes two importantly different cases that are often conflated:

- **True equilibrium lock-in:** no agent can improve the outcome by better individual choice alone.
- **Contingent coordination failure:** the bad outcome persists because of confusion, poor information, weak learning, or temporary noise; here, better decisions or clearer communication may suffice.

The diagnostic test is an extreme-case one: imagine all agents are fully informed, instrumentally rational, and aware that everyone prefers the superior coordinated outcome. If the failure still persists because no one can safely initiate the transition, then the persistence is structural. If it would vanish under those conditions, then it was contingent.

So the deeper determinant is not time, culture, or preference alone, but **transition topology**: whether there exists a credible path from the bad equilibrium to the good one that some sequence of individually defensible moves can traverse. If every path requires an actor to absorb unrecoverable risk before others follow, persistence is the default. If the system supplies even a small bridge — insurance, signaling, threshold effects, trusted leadership, repeated games, side payments, norms, or institutions — spontaneous resolution becomes possible.

In short: coordination failure persists when the obstacle is an equilibrium barrier, and resolves when the system generates a mechanism that converts collective desirability into individually rational transition.

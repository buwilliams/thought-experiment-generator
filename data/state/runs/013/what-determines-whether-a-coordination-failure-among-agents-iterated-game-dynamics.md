## Conjecture

**Conjecture:** A coordination failure persists when the interaction structure makes defection from the coordinated solution effectively invisible, low-cost, or one-shot; it resolves spontaneously when agents expect to meet again under conditions where past behavior can be observed and responded to, so that even self-interested agents acquire a reason to shift toward mutually reinforcing strategies.

From an iterated-game perspective, the key variable is not “willingness to cooperate” in the moral sense, but the **effective shadow of the future**. Coordination failures are often stable equilibria of repeated misalignment: each agent keeps choosing the locally safe move because unilateral deviation toward the better collective pattern is punished in the short run. The question is therefore: can repeated interaction transform that short-run risk into a long-run gain?

A spontaneous resolution becomes likely when three structural conditions hold:

1. **Repetition:** agents expect enough future encounters that current sacrifice can be repaid later.
2. **Observability:** others can detect whether an agent attempted coordination or defected from it.
3. **Reciprocal responsiveness:** agents can condition future behavior on what they observed.

When these are present, a new dynamic appears: an agent can “test” cooperation, others can reward it by matching, and failures can be corrected through adaptive reciprocity. Coordination is no longer a one-shot leap but an accumulative process of mutual prediction. In systems terms, repeated play creates a positive feedback loop: successful coordination increases trust in future coordination, which increases the payoff of coordinating again.

By contrast, failure persists indefinitely when the horizon is short, the population is transient, actions are noisy or hidden, or agents cannot target responses to particular past behavior. In those cases, the individually rational strategy is to stick with the existing failed convention, because no one can reliably recover the cost of moving first. The system then traps itself: everyone waits for evidence that others will coordinate, but the structure prevents such evidence from becoming credible.

An extreme-case test clarifies the point. If interaction is literally one-shot, spontaneous resolution is nearly impossible unless some external focal mechanism already aligns expectations. If interaction is indefinitely repeated with perfect memory and clear signals, many coordination failures become fragile, because once a cooperative pattern starts, reciprocity can lock it in.

So the deepest determinant is not agents’ character but whether the game lets **history matter**. Coordination failure persists when the structure erases history; it resolves when the structure converts history into reputational leverage.
